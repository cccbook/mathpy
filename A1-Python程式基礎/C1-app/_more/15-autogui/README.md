# pyautogui

pyautogui 執行時可能會控制你的滑鼠鍵盤到處亂跑，所以必須先學會如何跳出。

## 先學會如何跳出

1. 快速將滑鼠移到任意角落
2. 按下 Ctrl-Alt-Del

## Install

```
$ pip install pyautogui
```
